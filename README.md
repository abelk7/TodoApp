# TodoApp
Todolist app 

### Project Setup
* Maven project
* Java language
* Spring Boot: 2.7.15
* Jar packaging
* Java 11
* Dependencies :
  * Spring Web
  * Thymeleaf
  * Spring Data JPA
  * Validation
  * H2 Database
  * Lombok
  * Spring Boot DevTools
